Tutorials
=========

This chapter provides the full set of documentation targeting external web developers and start-ups. A getting started document is provided followed by an overview of the Hyperty Concept, the Messaging Framework used to support Hyperty's interaction, the higher level Data Synchronisation Reporter - Observer communication mechanism, as well as the Hyperty Trust and Security Model. Then, detailed Tutorials are provided to guide on the development of Hyperties, Applications and Message Nodes.
